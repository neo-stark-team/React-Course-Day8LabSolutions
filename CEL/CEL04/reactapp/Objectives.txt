React Component in Details

How to set default props in React.

defaultProps is a property on the componment class and can be used for undefine props. It cant be used for null props.

Program:
Let we create a CustomText class that returns label of text "Sample Label Text". Set the defaultProps property to the CustomText class color- red.
Call the CustomText componment from App Component with color props and without props. It will show the defaultProps property implemented.

Let the App component have h1 tag "Check defaultProps" and render CustomText four times.
