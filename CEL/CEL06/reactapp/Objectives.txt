React Component in Details

A detailed view of Component.

Program:
Add a separate file Arithmetic.js in src folder. 
This Arimetic component has add function and substract function.
The function choice is given by the operator prop as "+" or "-". 
Depends on the prop given the component will display the results in alert box.

Create class component Arithmetic with below requirements
1. class constructor with props and super props
2. state value declaration 
    firstNumber and secondNumber
3. return form element
    a. Two input values
    b. Two button with data-testid - formButtonAdd  and formButtonSubtract
    [Note : only one button should be visible at a time]

The parent component, App component renders the Arithmetic Function as below:
<Arithmetic operator='+'/>
<Arithmetic operator='-'/>
